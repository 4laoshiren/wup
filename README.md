# wup

A CLI tool that checks and updates packages across **npm** and **scoop** in one command — no need to remember which manager owns what.

## Usage

Check for outdated packages (interactive):

```bash
npx winup
```

Update a specific package:

```bash
npx winup <package-name>
```

Or install globally:

```bash
npm install -g winup
wup
wup <package-name>
```
